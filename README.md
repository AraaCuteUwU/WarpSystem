# WarpSystem

# Features
- Warps are registered as commands
- Warps can require permissions
- Warps are easy to setup

# Commands
- /warp (see all warps name) 
- /warp <warpName>
- /delwarp <warpName>
- /setwarp <warpName>

# Permissions
- setwarp.cmd
   default: op
- delwarp.cmd:
   default:op
- warp.cmd
   default: true
